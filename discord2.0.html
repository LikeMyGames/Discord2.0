Discord 2.0


This is my Discord clone. If you want to look at the source code for the javascript, html, or css, you can go to this link https://github.com/LikeMyGames/Discord2.0 This Discord clone is built using plain HTML 5, plain Javascript, and plain CSS. The text and video system are powered by the PeerJS library. https://peerjs.com/ All of the icons that are used in the project except for the ones that have image files are from https://fonts.google.com/icons


To use this application who can either create an account, use an existing account, or use without an accout.


If you create an account, it sends a put request to my custom built server, Discord2.0 API.
//Currently this feature is not working. Coming soon hopefully.


If you already have an account in the API, all you have to do is input the user name and password assosiated with the account. Any DM conversations that you had already started will be loaded and any servers will be loaded as well. All previous messages in DM conversations will be loaded when the DM conversation is clickes on in the list.


If you want to use this app without an account, you can. But without an account, you can only have text, audio, or video chats if you have their id. The id is usually the user's username with a hashtag and a randomly generated 4 digit hexadecimal number. While not using an account, you cannot participate in servers. Furthermore, none of your DM conversations will be saved at all.
